# strings.h additions

Add these define placeholders to your active strings header.

Replace the numeric values with free string IDs that do not collide with existing ones.

```cpp
#define IDS_ITEM_PRISMARINE_SHARD        TODO_STRING_ID_1
#define IDS_ITEM_PRISMARINE_CRYSTALS     TODO_STRING_ID_2
#define IDS_ITEM_BANNER                  TODO_STRING_ID_3
#define IDS_ITEM_END_CRYSTAL             TODO_STRING_ID_4
#define IDS_ITEM_ELYTRA                  TODO_STRING_ID_5
#define IDS_ITEM_CHORUS_FRUIT            TODO_STRING_ID_6
#define IDS_ITEM_CHORUS_FRUIT_POPPED     TODO_STRING_ID_7
#define IDS_ITEM_BEETROOT_SEEDS          TODO_STRING_ID_8
#define IDS_ITEM_BEETROOT                TODO_STRING_ID_9
```
